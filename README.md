# Coding Cat 🐱

A cute pixel cat that watches you code in VS Code.

## Features

- 🐱 A pixel art cat sits at a laptop in your editor
- ⌨️ Cat reacts when you type
- 💾 Cat celebrates when you save
- 😸 Smooth animations

## Usage

Open Command Palette (`Cmd+Shift+P`) and run:
- `Coding Cat: Start 🐱` — Show the cat panel
- `Coding Cat: Stop` — Hide the cat

The cat automatically appears when VS Code starts!
